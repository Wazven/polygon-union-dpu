# Union-polygon testing

## How to Run DockerFile

Clone the project

```bash
  gh repo clone Wazven/polygon-union-dpu
```

Go to the project directory

```bash
  cd polygon-union-dpu
```

Build DockerFile

```bash
  docker build -t your-image-name .
```

Run DockerFile

```bash
  docker run -p 8000:8000 your-image-name
```
